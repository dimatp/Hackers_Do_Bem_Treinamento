# Glossário

| Sigla / Termo | Significado |
|---|---|
| **AD** | Active Directory — serviço de diretório da Microsoft para gerenciamento de identidades e políticas em redes Windows |
| **C2** | Command and Control — infraestrutura usada pelo atacante para comunicar e controlar sistemas comprometidos |
| **CTI** | Cyber Threat Intelligence — inteligência sobre ameaças cibernéticas |
| **DHCP** | Dynamic Host Configuration Protocol — protocolo que atribui endereços IP automaticamente a dispositivos na rede |
| **DLL** | Dynamic-Link Library — biblioteca de código compartilhado no Windows |
| **EDR** | Endpoint Detection and Response — solução de segurança para detecção e resposta a ameaças em endpoints |
| **GPO** | Group Policy Object — objeto de política de grupo no Active Directory |
| **IoC** | Indicator of Compromise — artefato observável que indica comprometimento de um sistema |
| **MAC** | Media Access Control — endereço físico único de uma interface de rede |
| **MIME** | Multipurpose Internet Mail Extensions — padrão que define tipos de conteúdo em comunicações de rede |
| **OUI** | Organizationally Unique Identifier — primeiros 3 octetos do MAC address, identifica o fabricante |
| **PCAP** | Packet Capture — arquivo de captura de tráfego de rede |
| **PE** | Portable Executable — formato de arquivo executável do Windows (.exe, .dll) |
| **SMB** | Server Message Block — protocolo de compartilhamento de arquivos e recursos em redes Windows |
| **SMTP** | Simple Mail Transfer Protocol — protocolo de envio de e-mail |
| **TTP** | Tactics, Techniques and Procedures — descrição do comportamento de um adversário |
| **VBA** | Visual Basic for Applications — linguagem de macro do Microsoft Office |
| **Dropper** | Malware cujo objetivo é instalar outro malware no sistema |
| **Kill Chain** | Modelo de análise de ataque em fases sequenciais (Lockheed Martin) |
| **MAC times** | Timestamps de Modified, Accessed e Created de um arquivo |
| **Payload** | Componente malicioso executado após a fase de entrega |
| **Spearphishing** | Phishing direcionado a um indivíduo ou organização específica |
| **Thread hijacking** | Técnica do Emotet de responder a e-mails legítimos com conteúdo malicioso |
